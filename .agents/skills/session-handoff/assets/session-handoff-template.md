## H. Suggested Starter Prompt

---

## Authoring notes

- Prefer concrete bullets over narrative recap
- Name files, symbols, commands, and artifacts explicitly
- Preserve failed attempts if they affect future choices
- Separate verified facts from assumptions and unknowns
- If something is missing, say so directly rather than implying certainty
